# 更新日志

## v1.7.0 (2025-03-20)

### 新功能
- **定时任务成功通知**：支持任务执行成功时发送通知，可在任务高级设置中开启
- **资源告警通知**：CPU/内存/磁盘使用率超过阈值时自动发送通知（每30分钟最多一次）
- **环境变量更新时间**：表格新增"更新时间"列，直观查看变量变动时间
- **版本号显示**：侧边栏标题旁显示当前版本号徽章
- **订阅日志详情**：拉取日志列表新增"查看"按钮，可查看完整日志内容
- **环境变量 API group 参数**：开放 API 文档补充 group 分组参数说明

### Bug 修复
- **脚本文件夹删除失败**：修复删除文件夹显示成功但实际未删除的问题（未传递 type=directory 参数）
- **脚本上传残留**：上传文件对话框关闭后再次打开不再显示上次文件
- **依赖状态不一致**：服务启动时自动校验已安装依赖的实际状态，容器重建后自动标记丢失的依赖
- **pip 安装报错**：修复容器中 /tmp 目录不存在导致的 FileNotFoundError
- **用户名显示**：修复刷新页面后右上角显示 "User" 而非实际用户名的问题
- **ql repo 分支识别**：修正命令参数解析顺序，与青龙面板格式一致
- **订阅日志不完整**：保存完整拉取流程日志（包含开始、结束、自动创建任务等信息）
- **TypeScript 编译错误**：修复 MonacoEditor 组件的空值检查问题

### 优化
- **批量操作确认**：定时任务批量运行/启用/禁用/删除操作增加二次确认
- **GitHub 镜像加速**：更新订阅管理中的 GitHub 加速地址为 gh.301.ee
- **脚本管理菜单美化**：树节点右键菜单添加图标和分割线
- **中断状态清理**：服务重启时自动将卡在 installing/removing 状态的依赖重置为失败
